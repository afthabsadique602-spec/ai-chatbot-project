📚 AI Bookstore Assistant Chatbot

An interactive chatbot web application that acts as a customer support assistant for an online bookstore.

 🚀 Features

 💬 Real-time chat interface
 📖 Book recommendations based on mood and preferences
 🧠 Smart response system
 🌐 Simple web interface using Flask
 ⚡ Fast and lightweight

 🛠️ Tech Stack

 Python
 Flask
 HTML, CSS, JavaScript

 ▶️ How to Run

1. Clone the repository
2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
python app.py
```

4. Open in browser:

```
http://127.0.0.1:5000
```
